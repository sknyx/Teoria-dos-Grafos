Aula prática - Biblioteca Networkx para grafos

A biblioteca NetworkX é um pacote python para a criação, manipulação e o estudo de estruturas, dinâmica e funções de grafos/redes complexas.

Neste tutorial aprenderemos o básico. Para começar é necessário instalar o networkx através do comando pip install networkx

Criação de grafos

In [155]:
import networkx as nx

G = nx.Graph()

Por definição, um grafo é uma coleção de nós (vertices) junto com pares de vértices, chamados arestas. Usando NetworkX, nós podem ser quaisquer objetos indexáveis, por exemplo: strings de texto, uma imagem, um objeto XML, outro grafo,..., dentre outros.

Exercício - Extra

Uma estrutura secundária (bidimensional) de uma sequência de RNA é de grande importância na área de bioinformática, pois auxilia na identificação de qual função tal molécula desempenha.

Uma sequência de RNA, extraída de um DNA é uma sequência de nucleotídeos do alfabeto = {a,c,t,g}, onde t pode ser substituído por u, caso seja um miRNA. Abaixo temos uma estrutura primária de uma sequência de RNA:

GGCUUCCUGGUACUUACCACAGUCUGUAUUCUUACACUGACUGUAUAGAAAGAGGAGGC

Sua estrutura secundária pode ser obtida de diversos métodos diferentes, por exemplo usando a ferramenta RNAFold. A resposta deste algoritmo é uma representação da estrutura secundária denominada Dot-bracked Notation:

GGCUUCCUGGUACUUACCACAGUCUGUAUUCUUACACUGACUGUAUAGAAAGAGGAGGC 
.(((((((....((....((((((((((....))))..))))))..))....)))))))

Note que essa representação pode ser visualizada como um grafo...

1) A partir de uma entrada composta de duas strings: a sequência de RNA e a notação em parênteses (Dot-bracked Notation), crie um programa que exiba uma representação em grafos de sua estrutura secundária correspondente. Obs.: A entrada pode ser feita diretamente no código fonte:

 seq  = GGCUUCCUGGUACUUACCACAGUCUGUAUUCUUACACUGACUGUAUAGAAAGAGGAGGC 
 dbn  = .(((((((....((....((((((((((....))))..))))))..))....)))))))

Já a saída deve ser uma imagem no formato .png.

2) A partir de 1), altere o código para que os loops internos sejam coloridos de cor diferente do restante da estrutura secundária. Loops internos são porções não pareadas que estão dentro de stems (regiões pareadas contíguas).
