Aula prtica - Biblioteca Networkx para grafos

A biblioteca NetworkX um pacote python para a criao, manipulao e o estudo de estruturas, dinmica e funes de grafos/redes complexas.

Neste tutorial aprenderemos o bsico. Para comear necessrio instalar o networkx atravs do comando pip install networkx

Criao de grafos

In [155]:
import networkx as nx

G = nx.Graph()

Por definio, um grafo uma coleo de ns (vertices) junto com pares de vrtices, chamados arestas. Usando NetworkX, ns podem ser quaisquer objetos indexveis, por exemplo: strings de texto, uma imagem, um objeto XML, outro grafo,..., dentre outros.

Exerccio - Extra

Uma estrutura secundria (bidimensional) de uma sequncia de RNA de grande importncia na rea de bioinformtica, pois auxilia na identificao de qual funo tal molcula desempenha.

Uma sequncia de RNA, extrada de um DNA uma sequncia de nucleotdeos do alfabeto = {a,c,t,g}, onde t pode ser substitudo por u, caso seja um miRNA. Abaixo temos uma estrutura primria de uma sequncia de RNA:

GGCUUCCUGGUACUUACCACAGUCUGUAUUCUUACACUGACUGUAUAGAAAGAGGAGGC

Sua estrutura secundria pode ser obtida de diversos mtodos diferentes, por exemplo usando a ferramenta RNAFold. A resposta deste algoritmo uma representao da estrutura secundria denominada Dot-bracked Notation:

GGCUUCCUGGUACUUACCACAGUCUGUAUUCUUACACUGACUGUAUAGAAAGAGGAGGC 
.(((((((....((....((((((((((....))))..))))))..))....)))))))

Note que essa representao pode ser visualizada como um grafo...

1) A partir de uma entrada composta de duas strings: a sequncia de RNA e a notao em parnteses (Dot-bracked Notation), crie um programa que exiba uma representao em grafos de sua estrutura secundria correspondente. Obs.: A entrada pode ser feita diretamente no cdigo fonte:

 seq  = GGCUUCCUGGUACUUACCACAGUCUGUAUUCUUACACUGACUGUAUAGAAAGAGGAGGC 
 dbn  = .(((((((....((....((((((((((....))))..))))))..))....)))))))

J a sada deve ser uma imagem no formato .png.

2) A partir de 1), altere o cdigo para que os loops internos sejam coloridos de cor diferente do restante da estrutura secundria. Loops internos so pores no pareadas que esto dentro de stems (regies pareadas contguas).
